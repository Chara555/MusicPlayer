## 运行方法

1.预先准备好node.js运行工具

安装教程

[2024最新版Node.js下载安装及环境配置教程【保姆级】_nodejs下载-CSDN博客](https://blog.csdn.net/WHF__/article/details/129362462)

克隆项目

git clone https://github.com/Chara555/MusicPlayer.git
cd MusicPlayer

安装依赖

npm install

启动开发环境

npm run dev
